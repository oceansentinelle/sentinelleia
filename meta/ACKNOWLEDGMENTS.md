## English

## Usage

Cette mention doit apparaître dans tous les livrables Ocean Sentinel relevant de publication ou communication :

- Rapports PDF (footer ou section Remerciements)
- Dashboard web (footer + page À propos)
- Présentations / posters (slide Crédits)
- Repository GitHub public (README.md)
- Articles scientifiques (section Acknowledgments)
